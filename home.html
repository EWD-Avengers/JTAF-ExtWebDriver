Extensions for WebDriver

Extensions for WebDriver improves upon the powerful WebDriver API with robust features that keep your browser automation running smoothly. Widget library, improved session management, extended functions and more!

Learn more »

View project on GitHub »

Extended API

The clean and powerful WebDriver API has been extended to bring even more. ExtWebDriver provides useful API for writing UI automation and you can easily extend for your needs.

Widget Library

There is no need to rewrite automation code to handle HTML widgets. ExtWebDriver provides implementations for popular HTML widgets such as Table, Button, CheckBox, RadioButton, Select and more.

Session Factory

With simple configuration, you can create WebDriver browser sessions to suit your test needs. ExtWebDriver provides session management capabilities to ease development and execution.

GUI Properties

You can parameterize your XPaths, CSSSelectors, and other locator types for maximum reusability. This makes your automation code maintenance easy.

Multiple Browsers

Run your automated tests against your favorite browsers such as Chrome, Firefox, IE. All the browsers supported by WebDriver are available for you to use out of the box.

Open Source

This project is developed and maintained by the Open Source community on GitHub. You can use ExtWebDriver for free and easily extend for your needs.
